# 개발 보고서 - WordCard 시각적 개선

## 개요
WordCard의 사용자 경험을 극대화하기 위해 다양한 시각적 인터랙션 및 애니메이션을 추가하였습니다.

## 주요 변경 사항

### 1. 3D Tilt 효과 및 호버 인터랙션
- **동적 틸트(Tilt)**: 마우스 위치에 따라 카드가 3D로 기울어지는 효과를 구현했습니다.
- **감도 최적화**: 카드 앞면(Short Content)은 7도, 뒷면(Long Content)은 4도로 기울기 각도를 차등 적용하여 긴 텍스트의 가독성을 확보했습니다.
- **모바일 대응**: 터치 스크린에서의 Sticky Hover 문제를 방지하기 위해 CSS Media Query(`hover: hover`)와 JS 가드 코드를 적용하여 마우스 환경에서만 작동하도록 제한했습니다.

### 2. Aurora Glow & Spotlight
- **오로라 조명**: 마우스를 따라다니는 부드러운 조명 효과를 추가했습니다.
    - **앞면**: 차분한 파란색(Blue) 틴트
    - **뒷면**: 밝은 흰색(White) 글로우
- **Seamless Transition**: 카드가 뒤집히는(Flip) 동안에도 조명의 좌표와 상태를 유지하여 깜빡임 없는 색상 전환을 구현했습니다.

### 3. Interactive Border Glow
- **Masking 기법**: CSS `mask-composite`를 활용하여 카드 테두리에만 빛이 맺히는 고급스러운 효과를 구현했습니다.
- **아티팩트 수정**: `inset: 0.5px` 조정 및 `overflow-hidden` 적용을 통해 모서리 부분의 렌더링 깨짐 현상을 해결했습니다.

### 4. UI/UX 개선
- **AI 테마 버튼**: 단어 생성 UI에 그라디언트 및 Shimmer 애니메이션이 적용된 버튼을 도입하여 AI 기능을 강조했습니다.
- **Sparkles 아이콘**: AI 생성 기능을 상징하는 아이콘을 추가했습니다.
- **Slide Down 애니메이션**: 새로운 단어가 추가될 때 부드럽게 미끄러져 내려오는 트랜지션을 적용했습니다.
- **iOS 아이콘**: 홈 화면 추가 시 사용할 `apple-touch-icon`을 생성 및 적용했습니다.

### 5. Vercel을 통한 웹사이트 배포

## 기술적 세부 사항
- **CSS Variable & State**: React `useState`로 마우스 좌표를 관리하고, 인라인 스타일로 실시간 반영하여 퍼포먼스를 최적화했습니다.
- **Gradient Masking**: 복잡한 테두리 글로우 효과를 위해 `radial-gradient`와 `mask-image`를 결합했습니다.

## 잔여 과제 / 참고 사항
- 모바일(iOS)에서의 터치 인터랙션은 스크롤 제스처와의 충돌 문제로 현재 비활성화 상태입니다. 추후 제스처 라이브러리 도입 검토가 가능합니다.
