## TOEFL Mode Enhancements

- [ ] **분야별 평가 점수 및 피드백 도출**
  - 사용자가 TOEFL 모드(`toefl-complete`, `toefl-build`)에서 선택한 주제 분야(Topic Set)별로 정답률·점수를 누적 추적하고, 학습 완료 리포트와 대시보드에서 가시화한다.
  - **시각화: 수평 bar chart**
    - 분야 라벨을 좌측, 점수/정답률을 가로 막대로 표현
    - 각 막대 끝에 절대 수치(예: 78% · 12/15) 표기
    - 막대 색상은 디자인 시스템 토큰(`bg-accent-500`, `bg-brand-500`, `bg-warning-500`) 사용 — 강·중·약 구간(0–60 / 60–80 / 80–100)별 분기
    - 분야가 많아질 경우 상위 N개만 노출하는 필터 또는 스크롤 패널 도입
    - 모바일에서는 막대 너비를 컨테이너 폭 기반으로, 데스크탑에서는 고정 그리드로 정렬
  - 분야별 약점에 대한 AI 코멘트(개선점·다음 학습 제안) 포함.
  - 데이터 적재 위치는 추후 결정(localStorage vs 백엔드). 우선 localStorage 누적 → 백엔드 마이그레이션 가능하도록 service 계층에 isolation.
